Research Interests

My Ph.D research mainly focuses on two main topics:

  • Robot Perception, Planning and Control

    I have been working on two medical robotic systems that use multi-sensors fusion and calibration to improve the efficiency of image-guided surgery and robotic automation. The first system is a robotic laser system that can reconstruct a high-fidelity point cloud of biological tissue surface and achieve vision-based laser ablation. The second system is a stereovision-guided laser system that can achieve human-in-the-loop image-guided surgery. For the robot planning, I have been working on a surgical planning problem by which a laser orientation planner is designed to achieve a pre-defined surgical objective, such as avoid overcutting the healthy tissue. A laser-tissue geometric model is developed and the parameters are estimated by data-driven methods. For the robot control, I have been working on a medical robotic system to track and reconstruct the 3D contour of a peripheral vein beneath the surface skin of human arm, with the aim of automated intravenous insertion. This system leverages multiple sensors (force sensor, 3D vision with RGB-D camera, and ultrasound probe) to achieve vision-based control of the UR5 robotic arm for smooth and robust movement on the surface tissue.

    Courses in Robotics and Vision: Introduction to Robotics, Computer Vision, Numerical Optimization, Distributed Robotics, Advanced Robotic Design, Linear System Theory, Model Predictive Control, Digital Image Processing, Computational Photography.

  • Machine Learning for Medical Data Analysis

    I have been working on three machine projects that use machine learning tools for healthcare and translational medicine problems. The first project aims to build a machine learning pipeline to achieve automated tumor classification by using the 1D Fluorescence signal. The second project uses Transfer Learning for multi-class Skin Lesion Classification. The third project focuses on using statistical machine learning algorithms (SVM, KNN) for automated vascular anomalies classification,

    Courses in Machine Learning: Numerical Analysis, Theory and Algorithm in Machine Learning, Artificial Intelligence, Reinforcement Learning, Statistical Method and Experimental Design, Natural Language Processing, Data Analysis and Computation, Information Theory.
